This page is about Loops in JS

There are four types of loops in JS. (1) for Loop, (2) infinite Loop, (3) while Loop, (4) do while Loop.

# for Loop:This is the for loop's syntex: 

for (let i = 1; i<= 5; i++){
    console.log("Abid Shahrier");
} <!-- this is an alternative way to write (i++) statement: (i = i + 1) but generally we write this (i++)

After writing for keyword we add parenthesis. Between the parenthesis we add our loop's three statement. (let i = 1;) is our first complete statement, it is called initializing statement. It sets the value of "i" as 1 in starting. (i <= 5;) is our second statement which is called stopping condition. When the value of i will be greater then 5 this statment will stop the loop. It checks the value of i each time before the loop run. (i++) is orur third statement that is called updation condition which updates i each time the loop run.

# Note: (i) won't exist outside this for loop. Because it is defined only in the block of for loop.

# infinite Loop: When a loop's stopping condition never return false then the loop turns into a infinite loop which never ends.

# while Loop: Which thing we can do by for loop exactly same thing we can do by while loop, it is only two different ways to write lops. Here is the syntex of while loop: 

let i = 0;
while (i <= 10) {
      console.log("Abid Shahrier");
      i++;
}

We write our stopping condition in parenthesis at while loop. In this case generally the work of initialization we do before loop and updation we do inside the loop. Remember this in the end of while loop there wiil be no semicolon.
